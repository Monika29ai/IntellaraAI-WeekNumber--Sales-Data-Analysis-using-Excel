****Project Title**

**Sales Dashboard Analysis Using Excel**

**Project Overview****

This project analyzes sales data using Excel Pivot Tables, Pivot Charts, and KPI metrics. The dashboard provides insights into sales performance across regions, product categories, and years.

KPIs
Total Sales
Profit Margin

Year-over-Year Growth

**Key Insights**
Central region recorded the highest sales.
Office Supplies was the top-selling category.
Sales increased consistently each year.
Apple Smart Phone was the best-selling product.

**Business Recommendations**
Maintain sufficient inventory in the Central region.
Focus marketing efforts on high-performing categories.
Monitor annual sales trends and adjust strategies accordingly.
